ShutterDev API

API documentation for the ShutterDev photo gallery backend.

Public

Publicly accessible endpoints for viewing photos.

GET /api/photos Get a paginated list of all photos

Parameters

Name In Description
limit query The number of photos to return. (Default: 20)
offset query The number of photos to skip for pagination. (Default: 0)
GET /api/photos/:id Get a single photo by its ID

Parameters

Name In Description
id path The numeric ID of the photo to retrieve.

Admin

Protected endpoints for managing photos. Requires Authorization: Bearer <token> header.

POST /api/admin/photos Upload a new photo

Request Body (multipart/form-data)

Name Type Description
image file The high-resolution image file to upload.
title string (Optional) The title of the photo.
description string (Optional) The description of the photo.
tags string A comma-separated list of tags (e.g., "bengaluru,night,palace").
PUT /api/admin/photos/:id Update a photo's details

Parameters

Name In Description
id path The numeric ID of the photo to update.

Request Body (application/json)

{
    "title": "My New Updated Title",
    "description": "A new description for this photo."
}
DELETE /api/admin/photos/:id Delete a photo

Parameters

Name In Description
id path The numeric ID of the photo to delete.
